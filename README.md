#Projet travail pratique réalisé par Isaac Béliveau

Développé avec Unreal Engine 5

##Crédits

Contenus des packs de Kenney

Mon jeu se déroule dans un vieux manoir, le joueur doit explorer celui-ci pour trouver la sortie secrète. Dans le premier niveau, le joueur est dans une chambre, il doit faire un parcours en sautant sur des objets du manoir qui flottent dans l’air pour ensuite trouver une clé dissimulée dans un tiroir pour déverrouiller la porte de la chambre qui mène au grenier. Le deuxième niveau se déroule dans le grenier du manoir, où le joueur cherche deux morceaux d’un médaillon. Une fois assemblé, ce médaillon ouvre une porte coulissante qui est une étagère renfermant la sortie secrète du manoir. L’ambiance sonore est composée d'orages et de planchers qui craquent. Dans mes deux niveaux, je vais utiliser l’asset Retro Medieval Kit, Retro Urban Kit et Furniture Kit.
